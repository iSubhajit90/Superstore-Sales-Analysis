# 🛒 Retail Business Performance Dashboard — Superstore Dataset

## 📌 Project Overview
Analysis of the Kaggle Superstore dataset (9,994 orders, 2014–2018) to uncover
sales trends, profit drivers, and customer segment performance using
Microsoft Excel and Power Query.

## 📊 Key Metrics Uncovered
- Total Sales: $2,297,201
- Total Profit: $286,397 (12.47% Profit Margin)
- Total Orders: 9,994 | Total Units: 37,873

## 🔍 Analysis Performed
- Data Cleaning & Transformation (Power Query)
- Segment-wise Profit Analysis (Consumer, Corporate, Home Office)
- Regional & State-level Sales Analysis (Top 10 States)
- Discount vs Profit Impact Analysis
- Monthly Sales Trend (2014–2017)
- Top 10 Profit-making & Loss-making Products
- KPI Dashboard with Slicers & Dynamic Charts

## 💡 Key Insights
- Consumer segment generated highest profit ($134,119)
- Discounts above 40% consistently led to negative profit margins
- Technology category had highest profit; Tables sub-category made losses
- Peak sales in Q4 (November–December) — seasonal demand pattern

## 🛠️ Tools Used
- Microsoft Excel (Advanced)
- Power Query
- Pivot Tables & Pivot Charts
- Slicers & Dynamic Dashboards
